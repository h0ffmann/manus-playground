# Consultoria para Mulheres Empreendedoras - Estrutura de Diretórios

Este arquivo descreve a estrutura de diretórios e arquivos do projeto.

```
consultoria-empreendedoras/
├── .github/                    # Configurações do GitHub
│   └── workflows/              # Workflows do GitHub Actions
│       └── deploy.yml          # Configuração de deploy para GitHub Pages
├── .gitignore                  # Arquivos e diretórios ignorados pelo Git
├── LICENSE                     # Licença MIT
├── README.md                   # Documentação principal do projeto
├── generate-report.js          # Script para geração direta de relatórios
├── index.html                  # Página principal do formulário
├── netlify.toml                # Configuração para deploy no Netlify
├── package.json                # Configuração do projeto Node.js
├── script.js                   # JavaScript principal do formulário
├── email-service.js            # Serviço de geração e envio de emails
├── styles.css                  # Estilos CSS do formulário
└── report-template.html        # Template HTML para relatórios
```
